# Restaurante-RayoControl - GA8-220501096-AA1-EV01

Evidencia: Desarrollar software a partir de la integración de sus módulos componentes.

Esta versión integra los módulos web de productos y pedidos del proyecto Restaurante-RayoControl usando Java, Spring Boot, Thymeleaf, JPA, Maven y PostgreSQL.

## Módulos integrados

- Productos: administración de productos del menú.
- Pedidos: registro de pedidos asociados a productos existentes.

## Ejecución

```bash
mvn clean package
mvn test
mvn spring-boot:run
```
